###  Feature
1. Font 변경
2. Background Color 변경
3. Text Color 변경
4. Text Size 변경
5. Custom Font 다운로드 및 추가

<br>


### Git Management

- master : 개발 부분 release
- develop : 개발

**Commit Message 양식은 아래의 규칙을 따른다.**  

```
1. 기능 구현 : 구현 기능 - 내용 
2. 버그 수정 : 수정 기능 변경 전 -> 변경 후
```
